# Guides techniques

Cette section documente l'installation, la configuration et l'intégration de MapStore2 dans geOrchestra.

Elle couvre :

- les prérequis d'installation et les points d'entrée de déploiement
- la configuration via le répertoire de données geOrchestra
- les sujets d'administration propres au composant
- les détails de développement local et d'intégration avec la sécurité
