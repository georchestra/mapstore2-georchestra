## À propos de cette documentation

La documentation officielle de MapStore maintenue par GeoSolutions reste un complément utile sur les fonctionnalités génériques du produit :
[mapstore2.readthedocs.io](https://mapstore2.readthedocs.io/).

geOrchestra n'utilise qu'une partie du produit MapStore complet. Certaines interfaces, certains écrans ou certains comportements peuvent donc différer de la documentation amont.
