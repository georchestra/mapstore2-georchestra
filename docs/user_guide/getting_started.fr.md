# Prendre en main

MapStore est un framework permettant de construire des applications WebGIS avec des outils avancés de visualisation cartographique. Dans geOrchestra, il sert de visualiseur principal pour la consultation et le partage des données.

Quand vous ouvrez l'application, les principaux points d'entrée sont généralement :

- le visualiseur cartographique pour l'exploration et l'analyse
- l'éditeur de contexte pour les administrateurs et les utilisateurs avancés

Les guides MapStore amont permettent de comprendre rapidement ces interfaces :

- [Exploring maps](https://mapstore.readthedocs.io/en/latest/user-guide/exploring-maps/)
- [Managing contexts](https://mapstore.readthedocs.io/en/latest/user-guide/managing-contexts/)

Le reste de ce guide utilisateur apporte le contexte spécifique à geOrchestra et aux usages embarqués.
